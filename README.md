## Introduction
Ce projet est un tableau de bord qui permet la visualisation et l'analyse des données.

## Architecture
Le système est construit sur une architecture microservices.

## Database Schema
Le schéma de la base de données comprend plusieurs tables principales:
- Utilisateurs
- Données
- Rapports

## Setup Instructions
1. Clonez le dépôt : `git clone https://github.com/SAGBO4/test_dashbord`
2. Installez les dépendances : `npm install`
3. Configurez la base de données dans le fichier `.env`
4. Démarrez le serveur : `npm start`
